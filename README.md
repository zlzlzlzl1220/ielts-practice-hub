# IELTS Practice Hub

An IELTS practice website with 2020-2026 topic trends, original IELTS-style practice tests, listening scripts, browser text-to-speech playback, answers, explanations and study tools.

This site does not reproduce official IELTS exam papers or copyrighted audio. It uses original practice materials and links to official sample resources.

Included study tools:

- Local progress dashboard
- Complete, favourite and review queue controls
- Mixed mock-test mode with countdown timer
- Writing draft pad with word count
- Vocabulary memory and manual rolling review
- Real dictionary pronunciation fallback for vocabulary

Total items:

- Listening tests: 105
- Reading tests: 105
- Writing tasks: 168
- Speaking topic sets: 105
- Vocabulary items: 3000

Deploy with GitHub Pages from the repository root.
